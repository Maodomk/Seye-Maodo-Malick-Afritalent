# AfriTalent

## Présentation

AfriTalent est une plateforme fictive de mise en relation entre des freelances tech africains et des entreprises à la recherche de talents qualifiés.

Le projet a été réalisé dans le cadre d'un projet de développement web afin de mettre en pratique les technologies HTML, CSS, Bootstrap, JavaScript et Git/GitHub.

---

## Fonctionnalités

- Page d'accueil moderne
- Liste de freelances
- Page des tarifs
- Présentation de l'entreprise
- Formulaire de contact
- Mode sombre
- Bouton retour en haut
- Compteurs animés
- Animations des sections
- Filtrage des freelances
- Design responsive

---

## Technologies utilisées

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Git
- GitHub
